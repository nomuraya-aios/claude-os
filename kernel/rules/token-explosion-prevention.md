# トークン爆発防止ルール

Claude Code 運用における3類型のトークン爆発を防ぐ。

## 爆発の3類型

| 類型 | 説明 | 具体例 |
|------|------|--------|
| 一撃重量 | 1回の呼び出しで大量トークンを消費 | 巨大ファイルを丸ごと渡す |
| 連続爆発 | ループ内でLLMを繰り返し呼び出す | for file in files; do claude -p ...; done |
| 終了しないプロセス | LLMがコンパクション後も処理を続ける | judge → 全件score2 → 次の件へ → 無限ループ |

## 防止策

- スクリプト内でLLMをループ呼び出しするコードを書かない
- バックグラウンドジョブではAnthropicAPI直接呼び出しを避ける
- `--max-turns` を必ず設定する
- 処理済みフラグを厳密に管理し、再実行フラグを作らない
